# Student Management System CLI

A professional command-line based Student Management System built using JavaScript and Node.js.

This project demonstrates clean code practices including:
- Object-Oriented Programming
- Separation of Concerns
- Input Validation
- Error Handling
- Logging
- Testing


## Features

### Student Management

- Add new students
- View all students
- Find student by ID
- Update student information
- Delete students
- Input validation


### Teacher Management

- Add new teachers
- View all teachers
- Find teacher by ID
- Update teacher information
- Delete teachers


### Classroom Management

- Create classrooms
- View classrooms
- Assign teachers to classrooms
- Add students to classrooms
